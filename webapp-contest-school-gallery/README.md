# 🎓 School Gallery - 우리학교 추억 보관소

## 📸 프로젝트 소개
학교의 소중한 순간들을 사진으로 기록하고 공유하는 웹 애플리케이션입니다.
Pinterest 스타일의 아름다운 갤러리로 추억을 영원히 보관하세요.

## ✨ 주요 기능

### 사용자 기능
- 📤 **사진 업로드**: 드래그 앤 드롭으로 간편하게 사진 업로드
- 🖼️ **갤러리 보기**: Pinterest 스타일의 Masonry 레이아웃
- 📅 **연도별 필터링**: 졸업년도별로 추억 탐색
- 🔍 **사진 상세보기**: 클릭하여 큰 화면으로 감상

### 관리자 기능
- ✅ **사진 검토 시스템**: 업로드된 사진 승인/거절
- 🛡️ **품질 관리**: 학교에 적합한 콘텐츠만 게시

## 🚀 시작하기

### 설치 없이 바로 실행
1. `index.html` 파일을 브라우저에서 열기
2. 바로 사용 가능!

### 관리자 접속
- 관리자 버튼 클릭
- 비밀번호: `admin123`

## 💻 기술 스택
- **Frontend**: Pure HTML, CSS, JavaScript
- **Storage**: LocalStorage (서버 불필요)
- **Design**: 반응형 웹 디자인
- **Animation**: CSS Animations

## 🎨 디자인 특징
- 🎯 **미니멀 디자인**: 깔끔하고 직관적인 UI
- 📱 **반응형**: 모바일, 태블릿, 데스크탑 완벽 지원
- 🎪 **부드러운 애니메이션**: 자연스러운 인터랙션
- 🌈 **Pinterest 스타일**: 트렌디한 Masonry 레이아웃

## 📂 파일 구조
```
webapp-contest-school-gallery/

└── README.md       # 프로젝트 문서
```

## 🔄 작동 방식

### 일반 사용자 플로우
1. 사진 업로드 버튼 클릭
2. 사진 선택 또는 드래그 앤 드롭
3. 업로드 제출
4. 관리자 승인 대기
5. 승인 후 갤러리에 표시

### 관리자 플로우
1. 관리자 버튼 클릭
2. 비밀번호 입력
3. 대기 중인 사진 목록 확인
4. 각 사진 승인 또는 거절
5. 승인된 사진은 즉시 갤러리에 반영

## 🌟 특별 기능
- **자동 연도 태깅**: 업로드 시점의 연도 자동 기록
- **필터링 시스템**: 연도별로 추억 검색
- **토스트 알림**: 사용자 액션에 대한 즉각적인 피드백
- **샘플 데이터**: 첫 실행 시 예시 사진 자동 생성
